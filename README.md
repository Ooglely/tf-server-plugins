## Requirements ##
- Sourcemod and Metamod


## Installation ##
1. Grab the latest release from the release page and unzip it in your sourcemod folder.
2. Restart the server or type `sm plugins load tf-server-plugins` in the console to load the plugin.
3. The config file will be automatically generated in cfg/sourcemod/

## Configuration ##
- You can modify the phrases in addons/sourcemod/translations/tf-server-plugins.phrases.txt.
- Once the plugin has been loaded, you can modify the cvars in cfg/sourcemod/tf-server-plugins.cfg.
